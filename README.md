# simple-cli

A minimal Rust CLI example distributed with cargo-dist.

## Install

### macOS / Linux

Use the installer script generated by `cargo-dist`:

```bash
curl --proto '=https' --tlsv1.2 -LsSf https://github.com/JCLee777/simple-cli/releases/latest/download/simple-cli-installer.sh | sh
```

If you want to install into a custom directory:

```bash
curl --proto '=https' --tlsv1.2 -LsSf https://github.com/JCLee777/simple-cli/releases/latest/download/simple-cli-installer.sh | INSTALL_PATH=/usr/local/bin sh
```

### Windows PowerShell

`cargo-dist` can also generate a PowerShell installer in GitHub Releases:

```powershell
powershell -ExecutionPolicy Bypass -c "irm https://github.com/JCLee777/simple-cli/releases/latest/download/simple-cli-installer.ps1 | iex"
```

### Manual download

You can also download the archive for your platform from:

```text
https://github.com/JCLee777/simple-cli/releases/latest
```

After extracting, put the `simple-cli` binary somewhere in your `PATH`.

## Usage

### Basic Commands
```bash
cargo run -- hello --name Rust
cargo run -- sum 12 30
```

If installed from a release, run:

```bash
simple-cli hello --name Rust
simple-cli sum 12 30
```

### Git Commands
```bash
simple-cli status          # Show working tree status
simple-cli log             # Show recent commits
simple-cli log -n 5        # Show last 5 commits
simple-cli branch          # List all branches
simple-cli branch-name     # Show current branch
```

All git commands support `--path <repo>` to specify a repository path.

## Build a release binary

```bash
cargo build --release
```

## Generate distribution artifacts with cargo-dist

```bash
dist generate
dist build --artifacts=local
```

## Notes

- `dist-workspace.toml` contains the generated cargo-dist configuration.
